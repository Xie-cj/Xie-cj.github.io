JavaScript

2021-01-01, 441 words, 2 min read

1995年，JavaScript问世。JavaScript的主要用途是代替服务端进行数据验证，在数据不符合要求时，提示用户。因为当时的网速非常慢，加载网页十分消磨人们的耐心。网景公司希望在Navigator浏览器中加入一种可由客户端进行基本验证的脚本语言，JavaScript应运而生。

JavaScript历史

JavaScript期初初命名为LiveScript，为了赶上发布发布时间，网景与Java语言的发明者Sun公司（已被甲骨文收购）结盟，共同开发LiveScript。就在正式发布前，网景公司将LiveScript改名为JavaScript以蹭上Java的热度。

JavaScript三大组成部分

  1. ECMAScript
  2. DOM
  3. BOM

ECMAScript

ECMAScript是一种由Ecma国际（前身为欧洲计算机制造商协会，European Computer Manufacturers Association）通过ECMA-262标准化的脚本程序设计语言。这种语言在万维网上应用广泛，它往往被称为JavaScript或JScript，所以它可以理解为是JavaScript的一个标准,但实际上后两者是ECMA-262标准的实现和扩展。

简单来说，ECMAScript就是JavaScript的规范，虽然不同内核的浏览器里JavaScript的实现各不相同，但都遵循ECMAScript语言标准

DOM

文档对象模型（Document Object Mode， DOM）是W3C制定的标准接口规范，是一种处理HTML和XML文件的标准API。DOM通过创建表示文档的树，让开发者可以随心所欲的控制网页的内容和结构。使用DOM API可以轻松地删除、添加、替换、修改节点。

BOM

浏览器对象模型（Browser Object Model， BOM）用于支持访问和操作浏览器的窗口。使用BOM，开发者可以操控浏览器显示页面之外的部分。
